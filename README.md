# Reseller Sales KPI Dashboard (AdventureWorksDW2022)

This Power BI project analyzes reseller sales performance using the AdventureWorksDW2022 database.

##  Key Features
- Monthly profit and discount analysis  
- KPIs for profit growth and discount percentage  
- Trend visualization using line and bar charts  

##  Tools Used
- Power BI  
- SQL Server (AdventureWorksDW2022)  
- DAX for KPI calculations  

##  Insights
- Track monthly profit trends  
- Identify months with excessive discounts  
- Compare reseller performance over time
